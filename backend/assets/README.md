# Assets pour les ordonnances

Déposez ici les fichiers fournis par le praticien pour personnaliser les
documents PDF :

- `signature.png` : image transparente de la signature manuscrite.
- `stamp.png` : cachet ou logo du cabinet (PNG ou SVG).

Si ces fichiers sont absents, l'application utilise des signatures/cachets
vectoriels générés automatiquement pour ne pas bloquer la génération
d'ordonnances pendant la phase de développement. Remplacez simplement les
fichiers par vos visuels lorsque vous les recevrez.
