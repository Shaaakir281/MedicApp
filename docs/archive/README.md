# Archives documentaires

Ce répertoire conserve les documents historiques qui ne sont plus des sources de vérité actives.

- `plans/` : anciens sprints, roadmaps et statuts datés.
- `signatures/` : analyses et comptes rendus sur la signature électronique.
- `sms/` : analyses d'envoi SMS, UX et non-régression.
- `implementation/` : synthèses d'implémentations terminées.

Pour l'état courant, utiliser `../ETAT_PROJET.md` et `../ROADMAP.md`.
